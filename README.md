Sistema de Gestión de Inventario (CRUD)
Permite realizar operaciones fundamentales de inventario con una interfaz de usuario moderna y minimalista.

 Descripción del Sistema
El sistema permite gestionar un Productos.

Características principales:
Gestión Completa (CRUD): Registro, consulta, edición y eliminación de productos.

Búsqueda Dinámica: Filtrado en tiempo real por código o nombre del producto.

Validaciones Inteligentes: Control de códigos duplicados y persistencia de datos segura.



Cómo Ejecutar
Para poner en marcha el sistema, sigue estos pasos:

Requisitos:

Java JDK 17 o superior.

Maven (o un IDE como IntelliJ IDEA / NetBeans).

Instalación:

Clona o descarga el repositorio.

Abre el proyecto en tu IDE preferido.

Ejecución:

Localiza la clase HelloApplication (o la clase principal con el método main).

Ejecuta el proyecto. El sistema creará automáticamente la carpeta de datos necesaria si no existe.

  Archivo de Datos (Persistencia)
El sistema utiliza una persistencia ligera basada en archivos de texto  para garantizar que la información se conserve al cerrar la aplicación.

Ubicación: data/Productos.csv.
